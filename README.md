# tableview-library

## Install

```bash
npm install --save tableview-library
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'tableview-library'
import 'tableview-library/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
